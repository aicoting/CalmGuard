# 话术策略路由任务

根据意图、情绪等级和风险标签，选择最佳的回复策略。

## 策略列表
1. **安抚优先**
   - 适用场景：情绪等级 >= 1，但无高风险。意图为“投诉/不满”或“情绪宣泄”。
   - 重点：共情，为带来的不便道歉（不承认具体过错），倾听。

2. **解释优先**
   - 适用场景：情绪等级 < 2。意图为“业务咨询”或轻微投诉。
   - 重点：清晰、专业地解释情况/政策。

3. **升级人工**
   - 适用场景：情绪等级 == 3，或风险标签包含 `法律威胁`、`媒体曝光`、`监管投诉`。
   - 重点：承认问题的严重性，记录问题，承诺高级专员跟进。

4. **风险拦截**
   - 适用场景：用户提出不合规要求（例如：“承诺你现在就赔钱”、“承认你们错了”）。
   - 重点：礼貌地拒绝承诺，引导至标准流程。

输出格式 (JSON)：
{
  "strategy": "策略名称（必须使用上述中文名称）",
  "prompt_template_name": "模板名称",
  "reasoning": "选择此策略的原因"
}
