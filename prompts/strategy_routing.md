# 话术策略路由任务

根据意图、情绪等级和风险标签，选择最佳的回复策略。

## 策略列表
1. **热情导购**
   - 适用场景：意图为“商品咨询”，情绪平静。
   - 重点：详细介绍卖点，推荐优惠券，引导下单。

2. **安抚补偿**
   - 适用场景：情绪等级 >= 1，意图为“投诉/不满”或“物流问题”。
   - 重点：真诚道歉，解释原因（如物流延误），主动提出小额补偿（如赠送运费险、5元优惠券、优先发货）。

3. **标准售后**
   - 适用场景：意图为“售后/退换货”，无高风险。
   - 重点：引导用户按流程操作（如“点击订单详情-申请售后”），说明7天无理由规则。

4. **升级专员**
   - 适用场景：情绪等级 == 3，或包含 `职业打假`、`法律威胁`、`平台投诉` 标签。
   - 重点：停止自动化回复，告知将由高级专员/店长介入处理，避免激化矛盾。

5. **合规拒绝**
   - 适用场景：用户提出不合理要求（如“无理由仅退款”、“巨额赔偿”）。
   - 重点：温柔但坚定地解释平台规则，提供替代方案（如“帮您申请退货包运费”）。

输出格式 (JSON)：
{
  "strategy": "策略名称（必须使用上述中文名称）",
  "prompt_template_name": "模板名称",
  "reasoning": "选择此策略的原因"
}
