# 意图识别任务

分析用户的输入，并将意图分类为以下**其中一个**类别：

1. **投诉/不满** (Complaint/Dissatisfaction)：用户对服务、费用或产品不满意。
2. **情绪宣泄** (Emotional Venting)：用户主要是表达愤怒/沮丧，暂时没有明确的具体要求。
3. **业务咨询** (Business Inquiry)：用户询问功能、状态或操作方法。
4. **风险事件** (Risk Event)：用户报告账户冻结、资金被盗、系统错误导致损失。
5. **威胁/升级** (Threat/Escalation)：用户威胁采取法律行动、媒体曝光或要求经理介入。

输出格式 (JSON)：
{
  "intent": "类别名称（必须使用上述中文名称）",
  "confidence": 0.0-1.0,
  "reasoning": "简要说明"
}
