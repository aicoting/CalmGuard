# 意图识别任务

分析用户的输入，并将意图分类为以下**其中一个**类别：

1. **商品咨询** (Product Inquiry)：询问商品规格、库存、优惠活动、尺码推荐等。
2. **订单/物流查询** (Order/Logistics)：查询发货状态、物流进度、催发货、修改地址。
3. **售后/退换货** (After-sales/Return)：申请退货退款、换货、维修、反馈质量问题、7天无理由。
4. **投诉/不满** (Complaint/Dissatisfaction)：投诉客服态度、虚假宣传、物流太慢、包装破损。
5. **闲聊/其他** (Chit-chat/Other)：非业务相关的对话或无法识别的意图。

输出格式 (JSON)：
{
  "intent": "类别名称（必须使用上述中文名称）",
  "confidence": 0.0-1.0,
  "reasoning": "简要说明"
}
