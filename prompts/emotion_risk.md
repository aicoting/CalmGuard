# 情绪与风险评估任务

分析用户的输入，评估其情绪强度和潜在风险。

## 情绪等级 (0-3)
0: 平静 / 中性
1: 轻微不满 / 催促
2: 愤怒 / 失望 / 焦急
3: 极度愤怒 / 辱骂 / 威胁

## 风险标签 (Risk Tags)
识别是否包含以下任何一项：
- `职业打假`：提及广告法、工商局、假一赔三、三倍赔偿。
- `平台投诉`：提及投诉到平台官方、黑猫投诉、12315。
- `差评威胁`：威胁给差评、发朋友圈曝光。
- `退款纠纷`：坚持要求仅退款不退货（恶意倾向）。
- `法律威胁`：提及律师、起诉。

输出格式 (JSON)：
{
  "emotion_level": INT,
  "risk_tags": ["标签1", "标签2"],
  "risk_score": INT (0-100)
}
