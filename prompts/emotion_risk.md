# 情绪与风险评估任务

分析用户的输入，评估其情绪强度和潜在风险。

## 情绪等级 (0-3)
0: 平静 / 中性
1: 轻微不满 / 烦恼
2: 愤怒 / 沮丧
3: 敌对 / 激进 / 绝望

## 风险标签 (Risk Tags)
识别是否包含以下任何一项：
- `法律威胁`：提及律师、起诉、法院。
- `媒体曝光`：提及微博、抖音、报纸、“曝光”。
- `监管投诉`：提及银保监会、报警。
- `资金损失`：声称钱不见了或被冻结。
- `人身安全`：(极端情况) 提及自残 (触发立即升级)。

输出格式 (JSON)：
{
  "emotion_level": INT,
  "risk_tags": ["标签1", "标签2"],
  "risk_score": INT (0-100)
}
